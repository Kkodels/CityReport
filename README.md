## ğŸ¯ Kegunaan Aplikasi

### Untuk Masyarakat/Warga

CityReport membantu warga untuk:

- **Melaporkan Masalah dengan Mudah** - Cukup foto, pilih kategori, dan kirim. Lokasi otomatis terdeteksi dengan Geotag.

- **Memantau Perkembangan Laporan** - Ketahui status laporan Anda dari "Baru" hingga "Selesai" secara real-time.

- **Melihat Masalah di Sekitar** - Gunakan peta interaktif untuk melihat laporan-laporan dari warga lain di sekitar lokasi Anda.

- **Berpartisipasi dalam Prioritas** - Dukung laporan warga lain dengan voting untuk meningkatkan prioritas penanganan.

- **Membangun Kesadaran Bersama** - Lihat statistik dan laporan populer untuk memahami masalah utama di kota.

### Untuk Pemerintah/Admin

CityReport membantu admin untuk:

- **Dashboard Analitik** - Lihat semua laporan dengan analisis prioritas berdasarkan jumlah vote.

- **Peta Sebaran Masalah** - Visualisasi geotag untuk melihat hotspot permasalahan kota.

- **Prioritas Penanganan** - Laporan dengan vote tinggi otomatis naik ke daftar teratas.

- **Dokumentasi Lengkap** - Upload bukti penyelesaian untuk transparansi.
